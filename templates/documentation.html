Basketball Scrapers Documentation

This documentation provides an overview of the basketball scrapers API and its parameters.

Base URL

The base URL for the API is:

https://basketballdatabase.eu.pythonanywhere.com

API Parameters

Parameter Description Possible Values
f Target function to call. schedule, game, player
lpar League parameter. Specifies the league for which data is requested. See list of scrapers below.
spar Season parameter. Specifies the season for which data is requested. Depends on the league and season.
season Points to a specific season for matches. Depends on the league and season.
source Points to the match source. e.g., FIBA_Livesco
extid Points to a specific game or player. Depends on the league and data.

Example API Request

To retrieve the schedule for the NBA league for the 2022-2023 season:

https://basketballdatabase.eu.pythonanywhere.com?f=schedule&lpar=NBA&spar=2022-2023

Notes

  • Ensure that the lpar value matches the league you are querying.
  • The spar and season parameters must align with the league's available seasons.
  • Use the extid parameter to fetch specific game or player details.
